Financial Power BI Dashboard
Overview

This project presents an interactive Financial Analysis Dashboard built using Power BI to analyze financial performance, customer behavior, and transaction insights through dynamic visualizations and data modeling techniques.

The dashboard helps transform raw financial data into meaningful business insights that support decision-making and performance tracking.

Tools & Technologies
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
CSV Data Sources
Dataset

The project uses multiple CSV datasets including:

Customer Data
Cards Data
Transaction Data (sample dataset)

Due to GitHub file size limitations, a sample transaction dataset is included instead of the full dataset.

Dashboard Features
Financial performance overview
Customer analysis
Transaction analysis
KPI tracking
Interactive filtering & slicers
Dynamic visualizations
